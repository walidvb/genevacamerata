INTRODUCTION
------------

Current Maintainer: Jyotisankar Pradhan <jyotisankar.pradhan@gmail.com>

Ajax loading gif module helps site administrator to change the default ajax
loading icon to a custom one.
By allowing him to upload a new gif image via admin settings.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------
Administrator can set the custom gif image from the admin panel. 
For adding gif navigate to 
<yourdomain.com>/admin/ajax_loading_gif_config
