
-- SUMMARY --

The Views Vertical Tabs module adds vertical_tabs row plugin to views.

For a full description of the module, visit the project page:
  http://drupal.org/node/1906858

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/1906858


-- REQUIREMENTS --

views.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- USAGE --

* While creating view choose display format vertical_tabs. OR Update the display
  format of the existing view.

* After saving the display format choose the settings for vertical_tabs.
